# 🐄 Dairy Farm Hub

## 📌 About Project
Dairy Farm Hub is a full-stack web application built to manage dairy farm operations efficiently.

## 🚀 Features
- Manage farmers and vendors
- CRUD operations
- REST API integration
- MySQL database connection

## 🛠️ Tech Stack
- Node.js
- Express.js
- MySQL
- JavaScript

## ⚙️ How to Run
1. Clone the repository
2. Go to backend folder
3. Install dependencies:
   npm install
4. Start server:
   node Server.js

## 🔐 Environment Variables
Create a `.env` file in backend folder:
DB_HOST=127.0.0.1  
DB_USER=root  
DB_PASSWORD=your_password  
DB_NAME=dairyfarm_db
